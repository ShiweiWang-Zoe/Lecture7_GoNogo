# Lecture7_GoNogo
This repository is the Lecture7 assignment, which inlcudes a GoNogo code and necessary files to run it.
The GoNogo code creates a PyschoPy experiment that aims to measure inhibition functions. 
Participants are required to respond quickly to "Go" trials and inhibit responses during "No-Go" trials.
The task uses food images categorized by fat content:
  The low-fat foods are Go trials
  The high-fat foods are No-Go trials

In Go trials, participants see a green circle that says 'Go'. 
In NoGo trials, participants see a red circle that says 'No Go'.
